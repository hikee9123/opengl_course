## OpenGL

- 3D 그래픽을 화면에 표현하기 위해 필요한 기본적인 기능들을 모은 오픈 API
- SGI (실리콘 그래픽스) 사의 그래픽 라이브러리 GL의 오픈소스화
- 1992년 첫 버전 발표
- 버전업을 거쳐 현재 4.6 까지 업데이트됨

## OpenGL의 특징

- Low-level API
- Cross-language
- Cross-platform
- 개발 체계
  - Khronos Group은 API 스펙을 관리
  - 실제 OpenGL driver는 벤더에서 스펙에 맞게 개발
